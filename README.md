## Description

This project serves as DEX backend by wrapping common uniswap functions, like token balance check/ price quote/ swap, as common RESTFUL api. This can also provide functions for cross DEX/CEX arbitrage purpose.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
